# fun-router
A HTML custom element implementing the `<fun-router>` tag.

[ Add a nice image here ]

## Setup

### Installation
```
npm i fun-router
```

---

```Html
<script src="node_modules/fun-router/fun-router.js"></script>
```
or if you're bundling
```Javascript
import "fun-router";
// or
require("fun-router");
```


## Usage
```HTML
    <fun-router></fun-router>
```
